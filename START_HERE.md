# 🚀 Bắt đầu (1 phút)

**Cách dễ nhất** — không cần biết Python, terminal, gì cả:

1. Cài **Claude Code**: https://claude.ai/download
2. Mở folder này bằng Claude Code (`File → Open Folder → chọn folder này`)
3. Gõ vào chat: **`bắt đầu`** (hoặc `setup`, `chạy đi`...)

Claude sẽ tự kiểm tra Python, cài dependencies, hỏi bạn FAL_KEY, và làm hết mọi thứ.

---

## Bạn cần gì?

- **FAL_KEY** — đăng ký tại https://fal.ai/dashboard/keys (miễn phí), nạp credit ~$5
- **Claude Code** — miễn phí (yêu cầu tài khoản Anthropic)
- Khoảng 10 phút cho 1 level đầy đủ

## Nếu bạn là dev / đã có Python

Bỏ qua bước Claude Code:
```bash
python3 start.py
```

Xem README.md để biết chi tiết các phase.
